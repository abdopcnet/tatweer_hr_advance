# Workflow

## Bulk Leave Allocation

```
Create Bulk Leave Allocation
    ↓
Select Company, Leave Type, Dates
    ↓
Get Active Employees
    ↓
Review/Edit Table
    ↓
Create Bulk Leave Allocations
    ↓
Leave Allocations Created & Submitted
```

## Leave Management

```
Leave Type → Leave Policy → Policy Assignment
    ↓
Leave Allocation Created
    ↓
Employee Applies → Approver Reviews → Approved/Rejected
    ↓
Leave Ledger Updated
```

## Payroll Processing

```
Salary Structure → Assignment → Payroll Entry
    ↓
Calculate Salaries → Create Salary Slips
    ↓
Submit → Bank Entry → Journal Entry
```

## Employee Training

```
Training Request → Manager Comments → HR Comments
    ↓
Approved → Training Scheduled
```

## Experience Calculation

```
Daily Scheduled Task
    ↓
Get Active Employees
    ↓
Calculate Experience Years
    ↓
Update custom_total_experience_in_years
```
