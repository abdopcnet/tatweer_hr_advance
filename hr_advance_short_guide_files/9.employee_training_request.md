# Employee Training Request

## Overview
Employee Training Request DocType for HR Advance.

## Fields

### Main Fields

- `employee` (Link, Required, Options: Employee): Employee
- `employee_name` (Data): Employee Name (Fetched from: employee.employee_name)
- `request_date` (Date, Required): Request Date
- `training_course` (Link, Required, Options: Training Course Available): Training Course
- `reason_for_request` (Text, Required): Reason For Request
- `manager_comments` (Text Editor): Manager Comments
- `hr_comments` (Text Editor): HR Comments
- `department` (Link, Options: Department): Department (Fetched from: employee.department)

## Workflow

### Step 1: Create Document
1. Go to HR Advance → Employee Training Request → New
2. Fill required fields:
   - `employee`: Employee
   - `request_date`: Request Date
   - `training_course`: Training Course
   - `reason_for_request`: Reason For Request

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: format:ETR-{YY}-{####}

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
