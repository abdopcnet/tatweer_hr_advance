# Training Course Available

## Overview
Training Course Available DocType for HR Advance.

## Fields

### Main Fields

- `course_name` (Data, Required): Course Name
- `course_classification` (Link, Required, Options: Course Classification): Course Classification
- `disabled` (Check, Default: 0):  Disabled
- `has_certification` (Check, Default: 0): Has Certification
- `description` (Text Editor): Description
- `learning_objectives` (Text Editor): Learning Objectives

## Workflow

### Step 1: Create Document
1. Go to HR Advance → Training Course Available → New
2. Fill required fields:
   - `course_name`: Course Name
   - `course_classification`: Course Classification

### Step 2: Save/Submit
1. Review all fields
2. Save document

### Naming
- Auto-naming: field:course_name

## Important Notes
- Document must be saved to be effective
