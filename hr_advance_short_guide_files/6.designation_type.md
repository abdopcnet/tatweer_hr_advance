# Designation Type

## Overview
Designation Type DocType for HR Advance.

## Fields

### Main Fields

- `designation_type` (Data, Required): Designation Type
- `is_it_a_leadership_position` (Check, Default: 0):  Is it a leadership position?
- `basic_salary` (Currency): Basic Salary
- `responsibility_bonus` (Currency): Responsibility Bonus (Depends on: eval: doc.is_it_a_leadership_position ==1)
- `fuel_and_communications_allowance` (Currency): Fuel and Communications allowance (Depends on: eval: doc.is_it_a_leadership_position ==1)
- `distinction_bonus` (Currency): Distinction Bonus (Depends on: eval: doc.is_it_a_leadership_position ==0)

## Workflow
