# Exit Permission Request

## Overview
Exit Permission Request DocType for HR Advance.

## Fields

### Main Fields

- `employee` (Link, Required, Options: Employee): Employee
- `employee_name` (Data): Employee Name (Fetched from: employee.employee_name)
- `department` (Link, Options: Department): Department (Fetched from: employee.department)
- `company` (Link, Options: Company): Company (Fetched from: employee.company)
- `for_date` (Date, Required): For date
- `from_time` (Time, Required): From Time
- `to_time` (Time, Required): To Time
- `type` (Select, Required, Options: ظرف شخصي
مهمة عمل): Type
- `approver` (Link, Options: User): Approver (Fetched from: employee.leave_approver)
- `approver_name` (Data):  Approver Name (Fetched from: approver.full_name)
- `status` (Select, Required, Options: Open
Approved
Rejected
Cancelled, Default: Open): Status
- `السبب` (Small Text): السبب

### Read-only Fields

- `posting_date` (Date, Read-only): Posting Date
- `total_hours` (Float, Read-only): Total Hours

## Workflow

### Step 1: Create Document
1. Go to HR Advance → Exit Permission Request → New
2. Fill required fields:
   - `employee`: Employee
   - `for_date`: For date
   - `from_time`: From Time
   - `to_time`: To Time
   - `type`: Type
   - `status`: Status

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: format:EPR-{YYYY}-{MM}-{#####}

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
