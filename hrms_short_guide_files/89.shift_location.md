# Shift Location

## Overview
Shift Location DocType for HRMS.

## Fields

### Main Fields

- `location_name` (Data, Required): Location Name
- `checkin_radius` (Int): Checkin Radius
- `longitude` (Float): Longitude
- `geolocation` (Geolocation): Geolocation
- `latitude` (Float): Latitude

## Workflow

### Step 1: Create Document
1. Go to HRMS → Shift Location → New
2. Fill required fields:
   - `location_name`: Location Name

### Step 2: Save/Submit
1. Review all fields
2. Save document

### Naming
- Auto-naming: field:location_name

## Important Notes
- Document must be saved to be effective
