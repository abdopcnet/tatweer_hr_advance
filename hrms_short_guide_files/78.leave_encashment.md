# Leave Encashment

## Overview
Leave Encashment DocType for HRMS.

## Fields

### Main Fields

- `leave_period` (Link, Required, Options: Leave Period): Leave Period
- `employee` (Link, Required, Options: Employee): Employee
- `leave_type` (Link, Required, Options: Leave Type): Leave Type
- `encashment_date` (Date): Encashment Date
- `company` (Link, Required, Options: Company): Company (Fetched from: employee.company)
- `encashment_days` (Float): Encashment Days
- `payable_account` (Link, Options: Account): Payable Account (Depends on: pay_via_payment_entry)
- `pay_via_payment_entry` (Check, Default: 0): Pay Via Payment Entry
- `posting_date` (Date): Posting Date (Depends on: pay_via_payment_entry)
- `expense_account` (Link, Options: Account): Expense Account (Depends on: pay_via_payment_entry)
- `cost_center` (Link, Options: Cost Center): Cost Center

### Read-only Fields

- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `department` (Link, Read-only, Options: Department): Department (Fetched from: employee.department)
- `leave_allocation` (Link, Read-only, Options: Leave Allocation): Leave Allocation
- `leave_balance` (Float, Read-only): Leave Balance
- `encashment_amount` (Currency, Read-only): Encashment Amount
- `additional_salary` (Link, Read-only, Options: Additional Salary): Additional Salary
- `currency` (Link, Required, Read-only, Options: Currency): Currency (Depends on: eval:(doc.docstatus==1 || doc.employee))
- `actual_encashable_days` (Float, Read-only): Actual Encashable Days
- `paid_amount` (Currency, Read-only, Default: 0.0): Paid Amount (Depends on: pay_via_payment_entry)
- `status` (Select, Read-only, Options: Draft
Unpaid
Paid
Submitted
Cancelled): Status

## Workflow

### Step 1: Create Document
1. Go to HRMS → Leave Encashment → New
2. Fill required fields:
   - `leave_period`: Leave Period
   - `employee`: Employee
   - `leave_type`: Leave Type
   - `company`: Company

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: HR-ENC-.YYYY.-.#####

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
