# Bulk Salary Structure Assignment

## Overview
Bulk Salary Structure Assignment DocType for HRMS Payroll.

## Fields

### Main Fields

- `salary_structure` (Link, Required, Options: Salary Structure): Salary Structure
- `from_date` (Date, Required): From Date
- `income_tax_slab` (Link, Options: Income Tax Slab): Income Tax Slab (Depends on: salary_structure)
- `payroll_payable_account` (Link, Options: Account): Payroll Payable Account (Fetched from: .default_payroll_payable_account)
- `branch` (Link, Options: Branch): Branch
- `department` (Link, Options: Department): Department
- `designation` (Link, Options: Designation): Designation
- `employment_type` (Link, Options: Employment Type): Employment Type
- `grade` (Link, Options: Employee Grade): Employee Grade

### Read-only Fields

- `company` (Link, Required, Read-only, Options: Company): Company (Fetched from: salary_structure.company)
- `currency` (Link, Read-only, Options: Currency): Currency (Fetched from: salary_structure.currency) (Depends on: salary_structure)

## Workflow

### Step 1: Create Document
1. Go to HRMS Payroll → Bulk Salary Structure Assignment → New
2. Fill required fields:
   - `salary_structure`: Salary Structure
   - `from_date`: From Date

### Step 2: Save/Submit
1. Review all fields
2. Save document

## Important Notes
- Document must be saved to be effective
