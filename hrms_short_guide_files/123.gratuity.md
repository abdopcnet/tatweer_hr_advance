# Gratuity

## Overview
Gratuity DocType for HRMS Payroll.

## Fields

### Main Fields

- `employee` (Link, Required, Options: Employee): Employee
- `posting_date` (Date, Required): Posting date
- `current_work_experience` (Float, Default: 0): Current Work Experience
- `expense_account` (Link, Options: Account): Expense Account (Depends on: eval: !doc.pay_via_salary_slip)
- `mode_of_payment` (Link, Options: Mode of Payment): Mode of Payment (Depends on: eval: !doc.pay_via_salary_slip)
- `gratuity_rule` (Link, Required, Options: Gratuity Rule): Gratuity Rule
- `payable_account` (Link, Options: Account): Payable Account (Depends on: eval: !doc.pay_via_salary_slip)
- `cost_center` (Link, Options: Cost Center): Cost Center
- `pay_via_salary_slip` (Check, Default: 1): Pay via Salary Slip
- `payroll_date` (Date): Payroll Date (Depends on: pay_via_salary_slip)
- `salary_component` (Link, Options: Salary Component): Salary Component (Depends on: pay_via_salary_slip)

### Read-only Fields

- `company` (Link, Required, Read-only, Options: Company): Company (Fetched from: employee.company)
- `amount` (Currency, Required, Read-only, Default: 0): Total Amount
- `status` (Select, Read-only, Options: Draft
Unpaid
Paid
Submitted
Cancelled, Default: Draft): Status
- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `department` (Link, Read-only, Options: Department): Department (Fetched from: employee.department)
- `designation` (Data, Read-only): Designation (Fetched from: employee.designation)
- `paid_amount` (Currency, Read-only, Default: 0): Paid Amount (Depends on: eval:doc.pay_via_salary_slip == 0)

## Workflow

### Step 1: Create Document
1. Go to HRMS Payroll → Gratuity → New
2. Fill required fields:
   - `employee`: Employee
   - `posting_date`: Posting date
   - `gratuity_rule`: Gratuity Rule

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: HR-GRA-PAY-.#####

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
