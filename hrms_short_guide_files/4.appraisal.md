# Appraisal

## Overview
Appraisal DocType for HRMS.

## Fields

### Main Fields

- `naming_series` (Select, Required, Options: HR-APR-.YYYY.-): Series
- `employee` (Link, Required, Options: Employee): Employee
- `company` (Link, Required, Options: Company): Company
- `appraisal_cycle` (Link, Required, Options: Appraisal Cycle): Appraisal Cycle
- `employee_image` (Attach Image): Employee Image (Fetched from: employee.image)
- `appraisal_template` (Link, Options: Appraisal Template): Appraisal Template
- `reflections` (Text Editor): reflections
- `remarks` (Text): Remarks (Depends on: rate_goals_manually)

### Read-only Fields

- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `department` (Link, Read-only, Options: Department): Department (Fetched from: employee.department)
- `designation` (Link, Read-only, Options: Designation): Designation (Fetched from: employee.designation)
- `total_score` (Float, Read-only): Total Goal Score
- `self_score` (Float, Read-only): Total Self Score
- `avg_feedback_score` (Float, Read-only): Average Feedback Score
- `rate_goals_manually` (Check, Read-only, Default: 0): Rate Goals Manually
- `start_date` (Date, Read-only): Start Date
- `end_date` (Date, Read-only): End Date
- `goal_score_percentage` (Float, Read-only): Goal Score (%) (Depends on: eval: !doc.rate_goals_manually)

*... and 1 more fields*

### Child Tables

- `appraisal_kra` (Table): KRA vs Goals
  - Options: Appraisal KRA
- `goals` (Table): Goals
  - Options: Appraisal Goal
- `self_ratings` (Table): self_ratings
  - Options: Employee Feedback Rating

## Workflow

### Step 1: Create Document
1. Go to HRMS → Appraisal → New
2. Fill required fields:
   - `naming_series`: Series
   - `employee`: Employee
   - `company`: Company
   - `appraisal_cycle`: Appraisal Cycle

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document
