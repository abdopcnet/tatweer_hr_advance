# Salary Slip

## Overview
Salary Slip DocType for HRMS Payroll.

## Fields

### Main Fields

- `posting_date` (Date, Required): Posting Date
- `employee` (Link, Required, Options: Employee): Employee
- `employee_name` (Read Only, Required): Employee Name (Fetched from: employee.employee_name)
- `journal_entry` (Link, Options: Journal Entry): Journal Entry
- `company` (Link, Required, Options: Company): Company (Fetched from: employee.company)
- `letter_head` (Link, Options: Letter Head): Letter Head
- `start_date` (Date): Start Date
- `end_date` (Date): End Date
- `payroll_frequency` (Select, Options: 
Monthly
Fortnightly
Bimonthly
Weekly
Daily): Payroll Frequency
- `leave_without_pay` (Float): Leave Without Pay
- `total_working_hours` (Float): Total Working Hours
- `hour_rate` (Currency): Hour Rate
- `bank_name` (Data): Bank Name
- `bank_account_no` (Data): Bank Account No
- `deduct_tax_for_unclaimed_employee_benefits` (Check, Default: 0): Deduct Tax For Unclaimed Employee Benefits

## Workflow

### Step 1: Create Document
1. Go to HRMS Payroll → Salary Slip → New
2. Fill required fields:
   - `posting_date`: Posting Date
   - `employee`: Employee
   - `employee_name`: Employee Name
   - `company`: Company
   - `exchange_rate`: Exchange Rate

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
