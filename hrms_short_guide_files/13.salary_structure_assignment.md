# Salary Structure Assignment

## Overview
Salary Structure Assignment DocType for HRMS.

## Fields

### Main Fields

- `employee` (Link, Required, Options: Employee): Employee
- `employee_name` (Data, Read-only): Employee Name
- `department` (Link, Read-only, Options: Department): Department
- `designation` (Link, Read-only, Options: Designation): Designation
- `salary_structure` (Link, Required, Options: Salary Structure): Salary Structure
- `from_date` (Date, Required): From Date
- `company` (Link, Required, Options: Company): Company
- `base` (Currency, Options: currency): Base
- `variable` (Currency, Options: currency): Variable
- `amended_from` (Link, Read-only, Options: Salary Structure Assignment): Amended From
- `income_tax_slab` (Link, Options: Income Tax Slab): Income Tax Slab
- `currency` (Link, Required, Read-only, Options: Currency): Currency
- `payroll_payable_account` (Link, Options: Account): Payroll Payable Account
- `grade` (Link, Read-only, Options: Employee Grade): Grade
- `tax_deducted_till_date` (Currency, Options: currency): Tax Deducted Till Date
- `taxable_earnings_till_date` (Currency, Options: currency): Taxable Earnings Till Date

### Child Tables

- `payroll_cost_centers` (Table): Cost Centers
  - Options: Employee Cost Center

## Workflow

### Step 1: Create Document
1. Go to HRMS → Salary Structure Assignment → New
2. Fill required fields:
   - `employee`: Employee
   - `salary_structure`: Salary Structure
   - `from_date`: From Date
   - `company`: Company
   - `currency`: Currency

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
