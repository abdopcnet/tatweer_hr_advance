# Vehicle Log

## Overview
Vehicle Log DocType for HRMS.

## Fields

### Main Fields

- `naming_series` (Select, Required, Options: HR-VLOG-.YYYY.-): Series
- `license_plate` (Link, Required, Options: Vehicle): License Plate
- `employee` (Link, Required, Options: Employee): Employee (Fetched from: license_plate.employee)
- `model` (Read Only): Model (Fetched from: license_plate.model)
- `make` (Read Only): Make (Fetched from: license_plate.make)
- `date` (Date, Required): Date
- `odometer` (Int, Required): Current Odometer value 
- `fuel_qty` (Float): Fuel Qty
- `price` (Currency): Fuel Price
- `supplier` (Link, Options: Supplier): Supplier
- `invoice` (Data): Invoice Ref

### Read-only Fields

- `last_odometer` (Int, Required, Read-only): Last Odometer Value  (Fetched from: license_plate.last_odometer)

### Child Tables

- `service_detail` (Table): service_detail
  - Options: Vehicle Service

## Workflow

### Step 1: Create Document
1. Go to HRMS → Vehicle Log → New
2. Fill required fields:
   - `naming_series`: Series
   - `license_plate`: License Plate
   - `employee`: Employee
   - `date`: Date
   - `odometer`: Current Odometer value 

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: naming_series:

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
