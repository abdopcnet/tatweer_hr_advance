# Leave Policy

## Overview
Leave Policy defines leave types and annual allocations for employees.

## Fields

### Basic Information
- `title` (Data, Required): Policy name/title
- `leave_policy_details` (Table, Required): Child table with leave types

### Child Table (Leave Policy Detail)
- `leave_type` (Link, Required): Leave Type name
- `annual_allocation` (Float, Required): Annual leave days
- `max_consecutive_days` (Float): Maximum consecutive days allowed

## Workflow

### Step 1: Create Leave Policy
1. Go to HR → Leave Policy → New
2. Enter `title`
3. Add rows to `leave_policy_details` table

### Step 2: Add Leave Types
1. For each row, select `leave_type`
2. Enter `annual_allocation` (e.g., 30 days)
3. Set `max_consecutive_days` (optional)

### Step 3: Submit
1. Review all leave types
2. Submit document

## Important Notes
- Leave Policy must be submitted before assignment
- `annual_allocation` is used for earned leaves calculation
- Multiple leave types can be added to one policy
- Policy is assigned via Leave Policy Assignment
