# Appraisal Template

## Overview
Appraisal Template DocType for HRMS.

## Fields

### Main Fields

- `description` (Small Text): description
- `template_title` (Data, Required): Appraisal Template Title

### Child Tables

- `goals` (Table): KRAs
  - Options: Appraisal Template Goal
- `rating_criteria` (Table): Rating Criteria
  - Options: Employee Feedback Rating

## Workflow

### Step 1: Create Document
1. Go to HRMS → Appraisal Template → New
2. Fill required fields:
   - `template_title`: Appraisal Template Title

### Step 2: Save/Submit
1. Review all fields
2. Save document

### Naming
- Auto-naming: field:template_title

## Important Notes
- Document must be saved to be effective
