# Employee Onboarding

## Overview
Employee Onboarding manages the onboarding process for new employees.

## Fields

### Basic Information
- `job_applicant` (Link, Required): Link to Job Applicant
- `job_offer` (Link, Required): Link to Job Offer
- `company` (Link, Required): Company name
- `date_of_joining` (Date, Required): Joining date
- `boarding_begins_on` (Date, Required): Onboarding start date

### Employee Details
- `employee_name` (Data, Read-only): Fetched from job_applicant
- `employee` (Link, Read-only): Created employee (after completion)
- `department` (Link): Department assignment
- `designation` (Link): Designation
- `employee_grade` (Link): Employee grade
- `holiday_list` (Link): Holiday list

### Template
- `employee_onboarding_template` (Link): Template for onboarding activities

### Activities
- `activities` (Table): Onboarding tasks/activities
- `boarding_status` (Select, Read-only): Status (Pending/In Process/Completed)

## Workflow

### Step 1: Create Employee Onboarding
1. Go to HR → Employee Onboarding → New
2. Select `job_applicant`
3. Select `job_offer`
4. Select `company`
5. Set `date_of_joining` and `boarding_begins_on`
6. Select `employee_onboarding_template` (optional)

### Step 2: Submit
1. Submit document
2. System creates Project and Tasks automatically
3. `boarding_status` changes to "In Process"

### Step 3: Complete Activities
1. Go to Project created for onboarding
2. Complete all required tasks
3. System updates `boarding_status` to "Completed"

### Step 4: Create Employee
1. Click "Make Employee" button
2. System creates Employee record
3. `employee` field is populated

## Important Notes
- Employee Onboarding is optional but recommended
- Template can pre-populate activities
- All activities must be completed before creating employee
- Employee record is created from onboarding document
