# Salary Structure

## Overview
Salary Structure DocType for HRMS.

## Fields

### Main Fields

- `company` (Link, Required, Options: Company): Company
- `letter_head` (Link, Options: Letter Head): Letter Head
- `is_active` (Select, Required, Options: 
Yes
No, Default: Yes): Is Active
- `payroll_frequency` (Select, Options: 
Monthly
Fortnightly
Bimonthly
Weekly
Daily, Default: Monthly): Payroll Frequency
- `is_default` (Select, Read-only, Options: Yes
No, Default: No): Is Default
- `salary_slip_based_on_timesheet` (Check, Default: 0): Salary Slip Based on Timesheet
- `salary_component` (Link, Options: Salary Component): Salary Component
- `hour_rate` (Currency, Options: currency): Hour Rate
- `leave_encashment_amount_per_day` (Currency, Options: currency): Leave Encashment Amount Per Day
- `max_benefits` (Currency, Options: currency): Max Benefits (Amount)
- `total_earning` (Currency, Read-only, Options: currency): Total Earning
- `total_deduction` (Currency, Read-only, Options: currency): Total Deduction
- `net_pay` (Currency, Read-only, Options: currency): Net Pay
- `mode_of_payment` (Link, Options: Mode of Payment): Mode of Payment
- `payment_account` (Link, Options: Account): Payment Account
- `amended_from` (Link, Read-only, Options: Salary Structure): Amended From
- `currency` (Link, Required, Options: Currency): Currency

### Child Tables

- `earnings` (Table): Earnings
  - Options: Salary Detail
- `deductions` (Table): Deductions
  - Options: Salary Detail

## Workflow

### Step 1: Create Document
1. Go to HRMS → Salary Structure → New
2. Fill required fields:
   - `company`: Company
   - `is_active`: Is Active
   - `currency`: Currency

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
