# Employee Performance Feedback

## Overview
Employee Performance Feedback DocType for HRMS.

## Fields

### Main Fields

- `feedback` (Text Editor, Required): feedback
- `added_on` (Datetime, Required, Default: Now): Added On
- `employee` (Link, Required, Options: Employee): For Employee
- `appraisal` (Link, Required, Options: Appraisal): Appraisal
- `reviewer` (Link, Required, Options: Employee): Reviewer

### Read-only Fields

- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `total_score` (Float, Read-only): Total Score
- `reviewer_designation` (Link, Read-only, Options: Designation): Designation (Fetched from: reviewer.designation)
- `reviewer_name` (Data, Read-only): Reviewer Name (Fetched from: reviewer.employee_name)
- `department` (Link, Read-only, Options: Department): Department (Fetched from: employee.department)
- `designation` (Link, Read-only, Options: Designation): Designation (Fetched from: employee.designation)
- `appraisal_cycle` (Link, Read-only, Options: Appraisal Cycle): Appraisal Cycle (Fetched from: appraisal.appraisal_cycle)
- `user` (Link, Read-only, Options: User): User (Fetched from: reviewer.user_id)
- `company` (Link, Required, Read-only, Options: Company): Company (Fetched from: employee.company)

### Child Tables

- `feedback_ratings` (Table): Feedback Ratings
  - Options: Employee Feedback Rating

## Workflow

### Step 1: Create Document
1. Go to HRMS → Employee Performance Feedback → New
2. Fill required fields:
   - `feedback`: feedback
   - `added_on`: Added On
   - `employee`: For Employee
   - `appraisal`: Appraisal
   - `reviewer`: Reviewer

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: format:HR-PF-{YYYY}-{#####}

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
