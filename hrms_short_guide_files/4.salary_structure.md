# Salary Structure

## Overview
Salary Structure defines salary components and formulas for employees.

## Fields

### Basic Information
- `name` (Data, Auto): Structure name
- `company` (Link, Required): Company name
- `from_date` (Date, Required): Effective from date
- `payroll_frequency` (Select, Required): Payment frequency (Monthly/Weekly/etc.)

### Components
- `earnings` (Table): Earnings components (Basic, Allowances, etc.)
- `deductions` (Table): Deduction components (Tax, Insurance, etc.)

### Child Table Fields (Salary Component)
- `salary_component` (Link, Required): Component name
- `amount` (Currency): Fixed amount
- `formula` (Small Text): Calculation formula
- `condition` (Small Text): Conditional formula

## Workflow

### Step 1: Create Salary Structure
1. Go to Payroll → Salary Structure → New
2. Select `company`
3. Set `from_date`
4. Select `payroll_frequency`

### Step 2: Add Earnings
1. Add rows to `earnings` table
2. Select `salary_component` (e.g., Basic Salary)
3. Enter `amount` or `formula`
4. Add `condition` if needed

### Step 3: Add Deductions
1. Add rows to `deductions` table
2. Select `salary_component` (e.g., Income Tax)
3. Enter `amount` or `formula`
4. Add `condition` if needed

### Step 4: Submit
1. Review all components
2. Submit document

## Important Notes
- Salary Structure must be submitted before assignment
- Use formulas for dynamic calculations
- Components can be conditional based on employee data
- Structure is assigned via Salary Structure Assignment
