# Job Applicant

## Overview
Job Applicant DocType for HRMS.

## Fields

### Main Fields

- `applicant_name` (Data, Required): Applicant Name
- `email_id` (Data, Required): Email Address
- `status` (Select, Required, Options: Open
Replied
Rejected
Hold
Accepted): Status
- `job_title` (Link, Options: Job Opening): Job Opening
- `source` (Link, Options: Job Applicant Source): Source
- `source_name` (Link, Options: Employee): Source Name (Depends on: eval: doc.source=="Employee Referral" )
- `cover_letter` (Text): Cover Letter
- `resume_attachment` (Attach): Resume Attachment
- `phone_number` (Data): Phone Number
- `country` (Link, Options: Country): Country
- `resume_link` (Data): Resume Link
- `applicant_rating` (Rating): Applicant Rating
- `lower_range` (Currency): Lower Range
- `upper_range` (Currency): Upper Range
- `currency` (Link, Options: Currency): Currency

*... and 1 more fields*

### Read-only Fields

- `notes` (Data, Read-only): Notes
- `employee_referral` (Link, Read-only, Options: Employee Referral): Employee Referral

## Workflow

### Step 1: Create Document
1. Go to HRMS → Job Applicant → New
2. Fill required fields:
   - `applicant_name`: Applicant Name
   - `email_id`: Email Address
   - `status`: Status

### Step 2: Save/Submit
1. Review all fields
2. Save document

### Naming
- Auto-naming: HR-APP-.YYYY.-.#####
