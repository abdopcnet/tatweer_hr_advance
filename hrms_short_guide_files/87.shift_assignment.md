# Shift Assignment

## Overview
Shift Assignment DocType for HRMS.

## Fields

### Main Fields

- `employee` (Link, Required, Options: Employee): Employee
- `shift_type` (Link, Required, Options: Shift Type): Shift Type
- `company` (Link, Required, Options: Company): Company (Fetched from: employee.company)
- `start_date` (Date, Required): Start Date
- `end_date` (Date): End Date
- `status` (Select, Options: Active
Inactive, Default: Active): Status
- `shift_location` (Link, Options: Shift Location): Shift Location

### Read-only Fields

- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `department` (Link, Read-only, Options: Department): Department (Fetched from: employee.department)
- `shift_request` (Link, Read-only, Options: Shift Request): Shift Request
- `shift_schedule_assignment` (Link, Read-only, Options: Shift Schedule Assignment): Shift Schedule Assignment

## Workflow

### Step 1: Create Document
1. Go to HRMS → Shift Assignment → New
2. Fill required fields:
   - `employee`: Employee
   - `shift_type`: Shift Type
   - `company`: Company
   - `start_date`: Start Date

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: HR-SHA-.YY.-.MM.-.#####

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
