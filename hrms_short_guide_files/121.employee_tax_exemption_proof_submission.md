# Employee Tax Exemption Proof Submission

## Overview
Employee Tax Exemption Proof Submission DocType for HRMS Payroll.

## Fields

### Main Fields

- `employee` (Link, Required, Options: Employee): Employee
- `submission_date` (Date, Required): Submission Date
- `payroll_period` (Link, Required, Options: Payroll Period): Payroll Period
- `attachments` (Attach): Attachments
- `currency` (Link, Required, Options: Currency): Currency (Depends on: eval: doc.employee)

### Read-only Fields

- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `department` (Link, Read-only, Options: Department): Department (Fetched from: employee.department)
- `company` (Link, Required, Read-only, Options: Company): Company (Fetched from: employee.company)
- `total_actual_amount` (Currency, Read-only): Total Actual Amount
- `exemption_amount` (Currency, Read-only): Total Exemption Amount

### Child Tables

- `tax_exemption_proofs` (Table): Tax Exemption Proofs
  - Options: Employee Tax Exemption Proof Submission Detail

## Workflow

### Step 1: Create Document
1. Go to HRMS Payroll → Employee Tax Exemption Proof Submission → New
2. Fill required fields:
   - `employee`: Employee
   - `submission_date`: Submission Date
   - `payroll_period`: Payroll Period
   - `currency`: Currency

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: HR-TAX-PRF-.YYYY.-.#####

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
