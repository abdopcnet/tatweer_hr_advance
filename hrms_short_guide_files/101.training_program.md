# Training Program

## Overview
Training Program DocType for HRMS.

## Fields

### Main Fields

- `training_program` (Data, Required): Training Program
- `status` (Select, Options: Scheduled
Completed
Cancelled, Default: Scheduled): Status
- `company` (Link, Required, Options: Company): Company
- `trainer_name` (Data): Trainer Name
- `trainer_email` (Data): Trainer Email
- `supplier` (Link, Options: Supplier): Supplier
- `contact_number` (Data): Contact Number
- `description` (Text Editor, Required): Description

## Workflow

### Step 1: Create Document
1. Go to HRMS → Training Program → New
2. Fill required fields:
   - `training_program`: Training Program
   - `company`: Company
   - `description`: Description

### Step 2: Save/Submit
1. Review all fields
2. Save document

### Naming
- Auto-naming: field:training_program

## Important Notes
- Document must be saved to be effective
