# Training Result

## Overview
Training Result DocType for HRMS.

## Fields

### Main Fields

- `training_event` (Link, Required, Options: Training Event): Training Event
- `employee_emails` (Small Text): Employee Emails

### Child Tables

- `employees` (Table): Employees
  - Options: Training Result Employee

## Workflow

### Step 1: Create Document
1. Go to HRMS → Training Result → New
2. Fill required fields:
   - `training_event`: Training Event

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: HR-TRR-.YYYY.-.#####

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
