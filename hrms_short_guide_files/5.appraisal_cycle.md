# Appraisal Cycle

## Overview
Appraisal Cycle DocType for HRMS.

## Fields

### Main Fields

- `start_date` (Date, Required): Start Date
- `end_date` (Date, Required): End Date
- `description` (Text Editor): description
- `branch` (Link, Options: Branch): Branch
- `department` (Link, Options: Department): Department
- `designation` (Link, Options: Designation): Designation
- `company` (Link, Required, Options: Company): Company
- `cycle_name` (Data, Required): Cycle Name
- `kra_evaluation_method` (Select, Options: Automated Based on Goal Progress
Manual Rating, Default: Automated Based on Goal Progress): KRA Evaluation Method
- `final_score_formula` (Code): Final Score Formula (Depends on: calculate_final_score_based_on_formula)
- `calculate_final_score_based_on_formula` (Check, Default: 0): Calculate Final Score based on Formula

### Read-only Fields

- `status` (Select, Read-only, Options: Not Started
In Progress
Completed, Default: Not Started): Status

### Child Tables

- `appraisees` (Table): appraisees
  - Options: Appraisee

## Workflow

### Step 1: Create Document
1. Go to HRMS → Appraisal Cycle → New
2. Fill required fields:
   - `start_date`: Start Date
   - `end_date`: End Date
   - `company`: Company
   - `cycle_name`: Cycle Name

### Step 2: Save/Submit
1. Review all fields
2. Save document

### Naming
- Auto-naming: field:cycle_name

## Important Notes
- Document must be saved to be effective
