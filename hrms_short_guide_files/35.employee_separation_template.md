# Employee Separation Template

## Overview
Employee Separation Template DocType for HRMS.

## Fields

### Main Fields

- `title` (Data, Required): Title
- `company` (Link, Options: Company): Company
- `department` (Link, Options: Department): Department
- `designation` (Link, Options: Designation): Designation
- `employee_grade` (Link, Options: Employee Grade): Employee Grade

### Child Tables

- `activities` (Table): Activities
  - Options: Employee Boarding Activity

## Workflow

### Step 1: Create Document
1. Go to HRMS → Employee Separation Template → New
2. Fill required fields:
   - `title`: Title

### Step 2: Save/Submit
1. Review all fields
2. Save document
