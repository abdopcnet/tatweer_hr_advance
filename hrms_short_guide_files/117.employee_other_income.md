# Employee Other Income

## Overview
Employee Other Income DocType for HRMS Payroll.

## Fields

### Main Fields

- `employee` (Link, Required, Options: Employee): Employee
- `payroll_period` (Link, Required, Options: Payroll Period): Payroll Period
- `company` (Link, Required, Options: Company): Company
- `source` (Data): Source
- `amount` (Currency, Required): Amount

### Read-only Fields

- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)

## Workflow

### Step 1: Create Document
1. Go to HRMS Payroll → Employee Other Income → New
2. Fill required fields:
   - `employee`: Employee
   - `payroll_period`: Payroll Period
   - `company`: Company
   - `amount`: Amount

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: HR-INCOME-.######

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
