# Employee Skill Map

## Overview
Employee Skill Map DocType for HRMS.

## Fields

### Main Fields

- `employee` (Link, Options: Employee): Employee
- `employee_name` (Read Only): Employee Name (Fetched from: employee.employee_name)
- `designation` (Read Only): Designation (Fetched from: employee.designation)

### Child Tables

- `employee_skills` (Table): Employee Skills
  - Options: Employee Skill
- `trainings` (Table): Trainings
  - Options: Employee Training

### Step 2: Save/Submit
1. Review all fields
2. Save document
