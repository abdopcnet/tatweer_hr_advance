# Shift Assignment Tool

## Overview
Shift Assignment Tool DocType for HRMS.

## Fields

### Main Fields

- `action` (Select, Required, Options: Assign Shift
Assign Shift Schedule
Process Shift Requests, Default: Assign Shift): Action
- `company` (Link, Required, Options: Company): Company
- `shift_type` (Link, Options: Shift Type): Shift Type (Depends on: eval:doc.action === "Assign Shift")
- `status` (Select, Options: Active
Inactive, Default: Active): Status
- `start_date` (Date): Start Date
- `end_date` (Date): End Date
- `branch` (Link, Options: Branch): Branch
- `department` (Link, Options: Department): Department
- `designation` (Link, Options: Designation): Designation
- `employment_type` (Link, Options: Employment Type): Employment Type
- `shift_type_filter` (Link, Options: Shift Type): Shift Type
- `approver` (Link, Options: User): Approver
- `from_date` (Date): From Date
- `to_date` (Date): To Date
- `grade` (Link, Options: Employee Grade): Employee Grade

*... and 2 more fields*

## Workflow

### Step 1: Create Document
1. Go to HRMS → Shift Assignment Tool → New
2. Fill required fields:
   - `action`: Action
   - `company`: Company

### Step 2: Save/Submit
1. Review all fields
2. Save document

## Important Notes
- Document must be saved to be effective
