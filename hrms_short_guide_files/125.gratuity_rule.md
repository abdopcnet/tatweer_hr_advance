# Gratuity Rule

## Overview
Gratuity Rule DocType for HRMS Payroll.

## Fields

### Main Fields

- `disable` (Check, Default: 0): Disable
- `calculate_gratuity_amount_based_on` (Select, Required, Options: Current Slab
Sum of all previous slabs): Calculate Gratuity Amount Based On
- `applicable_earnings_component` (Table MultiSelect, Required): Applicable Earnings Component
- `work_experience_calculation_function` (Select, Options: Round off Work Experience
Take Exact Completed Years
Manual, Default: Round off Work Experience): Work Experience Calculation Method
- `total_working_days_per_year` (Float, Default: 365): Total working Days Per Year
- `minimum_year_for_gratuity` (Int): Minimum Year for Gratuity

### Child Tables

- `gratuity_rule_slabs` (Table): Current Work Experience
  - Options: Gratuity Rule Slab

## Workflow

### Step 1: Create Document
1. Go to HRMS Payroll → Gratuity Rule → New
2. Fill required fields:
   - `calculate_gratuity_amount_based_on`: Calculate Gratuity Amount Based On
   - `applicable_earnings_component`: Applicable Earnings Component

### Step 2: Save/Submit
1. Review all fields
2. Save document

### Naming
- Auto-naming: Prompt

## Important Notes
- Document must be saved to be effective
