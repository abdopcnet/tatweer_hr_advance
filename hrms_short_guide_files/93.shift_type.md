# Shift Type

## Overview
Shift Type DocType for HRMS.

## Fields

### Main Fields

- `start_time` (Time, Required): Start Time
- `end_time` (Time, Required): End Time
- `holiday_list` (Link, Options: Holiday List): Holiday List
- `determine_check_in_and_check_out` (Select, Options: Alternating entries as IN and OUT during the same shift
Strictly based on Log Type in Employee Checkin): Determine Check-in and Check-out
- `working_hours_calculation_based_on` (Select, Options: First Check-in and Last Check-out
Every Valid Check-in and Check-out): Working Hours Calculation Based On
- `working_hours_threshold_for_half_day` (Float): Working Hours Threshold for Half Day
- `working_hours_threshold_for_absent` (Float): Working Hours Threshold for Absent
- `begin_check_in_before_shift_start_time` (Int, Default: 60): Begin check-in before shift start time (in minutes)
- `late_entry_grace_period` (Int): Late Entry Grace Period (Depends on: enable_late_entry_marking)
- `early_exit_grace_period` (Int): Early Exit Grace Period (Depends on: enable_early_exit_marking)
- `allow_check_out_after_shift_end_time` (Int, Default: 60): Allow check-out after shift end time (in minutes)
- `enable_auto_attendance` (Check, Default: 0): Enable Auto Attendance
- `process_attendance_after` (Date): Process Attendance After
- `last_sync_of_checkin` (Datetime): Last Sync of Checkin
- `mark_auto_attendance_on_holidays` (Check, Default: 0): Mark Auto Attendance on Holidays

*... and 4 more fields*

## Workflow

### Step 1: Create Document
1. Go to HRMS → Shift Type → New
2. Fill required fields:
   - `start_time`: Start Time
   - `end_time`: End Time

### Step 2: Save/Submit
1. Review all fields
2. Save document
