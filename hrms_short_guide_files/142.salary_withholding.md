# Salary Withholding

## Overview
Salary Withholding DocType for HRMS Payroll.

## Fields

### Main Fields

- `employee` (Link, Required, Options: Employee): Employee
- `number_of_withholding_cycles` (Int, Required): Number of Withholding Cycles
- `posting_date` (Date, Required): Posting Date
- `from_date` (Date, Required): From Date
- `reason_for_withholding_salary` (Small Text): Reason for Withholding Salary

### Read-only Fields

- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `company` (Link, Read-only, Options: Company): Company (Fetched from: employee.company)
- `payroll_frequency` (Select, Read-only, Options: 
Monthly
Fortnightly
Bimonthly
Weekly
Daily): Payroll Frequency
- `to_date` (Date, Required, Read-only): To Date
- `date_of_joining` (Date, Read-only): Date of Joining (Fetched from: employee.date_of_joining)
- `relieving_date` (Date, Read-only): Relieving Date (Fetched from: employee.relieving_date)
- `status` (Select, Read-only, Options: 
Draft
Withheld
Released
Cancelled, Default: Draft): Status

### Child Tables

- `cycles` (Table): Cycles
  - Options: Salary Withholding Cycle

## Workflow

### Step 1: Create Document
1. Go to HRMS Payroll → Salary Withholding → New
2. Fill required fields:
   - `employee`: Employee
   - `number_of_withholding_cycles`: Number of Withholding Cycles
   - `posting_date`: Posting Date
   - `from_date`: From Date

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: format:SAL-WTH-{#####}

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
