# Payroll Entry

## Overview
Payroll Entry DocType for HRMS.

## Fields

### Main Fields

- `posting_date` (Date, Required, Default: Today): Posting Date
- `payroll_frequency` (Select, Options: 
Monthly
Fortnightly
Bimonthly
Weekly
Daily): Payroll Frequency
- `company` (Link, Required, Options: Company): Company
- `branch` (Link, Options: Branch): Branch
- `department` (Link, Options: Department): Department
- `designation` (Link, Options: Designation): Designation
- `number_of_employees` (Int, Read-only): Number Of Employees
- `validate_attendance` (Check, Default: 0): Validate Attendance
- `salary_slip_based_on_timesheet` (Check, Default: 0): Salary Slip Based on Timesheet
- `start_date` (Date, Required): Start Date
- `end_date` (Date, Required): End Date
- `deduct_tax_for_unclaimed_employee_benefits` (Check, Default: 0): Deduct Tax For Unclaimed Employee Benefits
- `deduct_tax_for_unsubmitted_tax_exemption_proof` (Check, Default: 0): Deduct Tax For Unsubmitted Tax Exemption Proof
- `cost_center` (Link, Required, Options: Cost Center, Default: :Company): Cost Center
- `project` (Link, Options: Project): Project
- `payment_account` (Link, Options: Account): Payment Account
- `amended_from` (Link, Read-only, Options: Payroll Entry): Amended From
- `salary_slips_created` (Check, Read-only, Default: 0): Salary Slips Created
- `salary_slips_submitted` (Check, Read-only, Default: 0): Salary Slips Submitted
- `bank_account` (Link, Options: Bank Account): Bank Account
- `currency` (Link, Required, Options: Currency): Currency
- `exchange_rate` (Float, Required): Exchange Rate
- `payroll_payable_account` (Link, Required, Options: Account): Payroll Payable Account
- `error_message` (Small Text, Read-only): Error Message
- `status` (Select, Read-only, Options: Draft
Submitted
Cancelled
Queued
Failed): Status
- `grade` (Link, Options: Employee Grade): Grade

### Child Tables

- `employees` (Table): employees
  - Options: Payroll Employee Detail

## Workflow

### Step 1: Create Document
1. Go to HRMS → Payroll Entry → New
2. Fill required fields:
   - `posting_date`: Posting Date
   - `company`: Company
   - `start_date`: Start Date
   - `end_date`: End Date
   - `cost_center`: Cost Center

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
