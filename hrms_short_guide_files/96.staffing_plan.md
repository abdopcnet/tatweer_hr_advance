# Staffing Plan

## Overview
Staffing Plan DocType for HRMS.

## Fields

### Main Fields

- `company` (Link, Required, Options: Company): Company
- `department` (Link, Options: Department): Department
- `from_date` (Date, Required): From Date
- `to_date` (Date, Required): To Date

### Read-only Fields

- `total_estimated_budget` (Currency, Read-only, Default: 0.00): Total Estimated Budget

### Child Tables

- `staffing_details` (Table): Staffing Details
  - Options: Staffing Plan Detail

## Workflow

### Step 1: Create Document
1. Go to HRMS → Staffing Plan → New
2. Fill required fields:
   - `company`: Company
   - `from_date`: From Date
   - `to_date`: To Date

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: prompt

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
