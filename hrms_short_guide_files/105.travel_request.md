# Travel Request

## Overview
Travel Request DocType for HRMS.

## Fields

### Main Fields

- `travel_type` (Select, Required, Options: 
Domestic
International): Travel Type
- `travel_funding` (Select, Options: 
Require Full Funding
Fully Sponsored
Partially Sponsored, Require Partial Funding): Travel Funding
- `travel_proof` (Attach): Copy of Invitation/Announcement
- `purpose_of_travel` (Link, Required, Options: Purpose of Travel): Purpose of Travel
- `details_of_sponsor` (Data): Details of Sponsor (Name, Location)
- `description` (Small Text): Any other details
- `employee` (Link, Required, Options: Employee): Employee
- `cell_number` (Data): Contact Number (Fetched from: employee.cell_number)
- `prefered_email` (Data): Contact Email (Fetched from: employee.prefered_email)
- `personal_id_type` (Link, Options: Identification Document Type): Identification Document Type
- `personal_id_number` (Data): Identification Document Number
- `passport_number` (Data): Passport Number (Fetched from: employee.passport_number)
- `cost_center` (Link, Options: Cost Center): Cost Center
- `name_of_organizer` (Data): Name of Organizer
- `address_of_organizer` (Data): Address of Organizer

*... and 1 more fields*

### Read-only Fields

- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `date_of_birth` (Date, Read-only): Date of Birth (Fetched from: employee.date_of_birth)
- `company` (Link, Read-only, Options: Company): Company (Fetched from: employee.company)

### Child Tables

- `itinerary` (Table): itinerary
  - Options: Travel Itinerary
- `costings` (Table): Costing
  - Options: Travel Request Costing

## Workflow

### Step 1: Create Document
1. Go to HRMS → Travel Request → New
2. Fill required fields:
   - `travel_type`: Travel Type
   - `purpose_of_travel`: Purpose of Travel
   - `employee`: Employee

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: HR-TRQ-.YYYY.-.#####

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
