# Training Event

## Overview
Training Event DocType for HRMS.

## Fields

### Main Fields

- `event_name` (Data, Required): Event Name
- `training_program` (Link, Options: Training Program): Training Program
- `event_status` (Select, Required, Options: Scheduled
Completed
Cancelled): Event Status
- `has_certificate` (Check, Default: 0): Has Certificate (Depends on: eval:doc.type == 'Seminar' || doc.type == 'Workshop' || doc.type == 'Conference' || doc.type == 'Exam')
- `type` (Select, Required, Options: Seminar
Theory
Workshop
Conference
Exam
Internet
Self-Study): Type
- `level` (Select, Options: 
Beginner
Intermediate
Advance): Level (Depends on: eval:doc.type == 'Seminar' || doc.type == 'Workshop' || doc.type == 'Exam')
- `company` (Link, Options: Company): Company
- `trainer_name` (Data): Trainer Name
- `trainer_email` (Data): Trainer Email
- `supplier` (Link, Options: Supplier): Supplier
- `contact_number` (Data): Contact Number
- `course` (Data): Course
- `location` (Data, Required): Location
- `start_time` (Datetime, Required): Start Time
- `end_time` (Datetime, Required): End Time

*... and 2 more fields*

### Child Tables

- `employees` (Table): Employees
  - Options: Training Event Employee

## Workflow

### Step 1: Create Document
1. Go to HRMS → Training Event → New
2. Fill required fields:
   - `event_name`: Event Name
   - `event_status`: Event Status
   - `type`: Type
   - `location`: Location
   - `start_time`: Start Time
   - `end_time`: End Time
   - `introduction`: Introduction

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: field:event_name

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
