# PWA Notification

## Overview
PWA Notification DocType for HRMS.

## Fields

### Main Fields

- `from_user` (Link, Options: User): From User (Fetched from: from_employee.user_id)
- `to_user` (Link, Options: User): To User (Fetched from: to_employee.user_id)
- `message` (Text Editor): Message
- `read` (Check, Default: 0): Read
- `reference_document_type` (Link, Options: DocType): Reference Document Type
- `reference_document_name` (Data): Reference Document Name

## Workflow

### Step 1: Create Document
1. Go to HRMS → PWA Notification → New

### Step 2: Save/Submit
1. Review all fields
2. Save document

### Naming
- Auto-naming: autoincrement

## Important Notes
- Document must be saved to be effective
