# Compensatory Leave Request

## Overview
Compensatory Leave Request DocType for HRMS.

## Fields

### Main Fields

- `employee` (Link, Required, Options: Employee): Employee
- `leave_type` (Link, Options: Leave Type): Leave Type
- `work_from_date` (Date, Required): Work From Date
- `work_end_date` (Date, Required): Work End Date
- `half_day` (Check): Half Day
- `half_day_date` (Date): Half Day Date (Depends on: half_day)
- `reason` (Small Text, Required): Reason

### Read-only Fields

- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `department` (Link, Read-only, Options: Department): Department (Fetched from: employee.department)
- `leave_allocation` (Link, Read-only, Options: Leave Allocation): Leave Allocation

## Workflow

### Step 1: Create Document
1. Go to HRMS → Compensatory Leave Request → New
2. Fill required fields:
   - `employee`: Employee
   - `work_from_date`: Work From Date
   - `work_end_date`: Work End Date
   - `reason`: Reason

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: HR-CMP-.YY.-.MM.-.#####

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
