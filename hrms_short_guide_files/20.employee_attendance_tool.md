# Employee Attendance Tool

## Overview
Employee Attendance Tool DocType for HRMS.

## Fields

### Main Fields

- `date` (Date): Date
- `department` (Link, Options: Department): Department
- `branch` (Link, Options: Branch): Branch
- `company` (Link, Options: Company): Company
- `status` (Select, Options: 
Present
Absent
Half Day
Work From Home): Status
- `late_entry` (Check, Default: 0): Late Entry
- `early_exit` (Check, Default: 0): Early Exit
- `shift` (Link, Options: Shift Type): Shift
- `employment_type` (Link, Options: Employment Type): Employment Type
- `designation` (Link, Options: Designation): Designation
- `employee_grade` (Link, Options: Employee Grade): Employee Grade
- `half_day_status` (Select, Options: Present
Absent): Status for Other Half

## Workflow

### Step 1: Create Document
1. Go to HRMS → Employee Attendance Tool → New

### Step 2: Save/Submit
1. Review all fields
2. Save document

## Important Notes
- Document must be saved to be effective
