# Attendance

## Overview
Attendance DocType for HRMS.

## Fields

### Main Fields

- `naming_series` (Select, Required, Options: HR-ATT-.YYYY.-): Series
- `employee` (Link, Required, Options: Employee): Employee
- `status` (Select, Required, Options: 
Present
Absent
On Leave
Half Day
Work From Home, Default: Present): Status
- `leave_type` (Link, Options: Leave Type): Leave Type (Depends on: eval:in_list(["On Leave", "Half Day"], doc.status))
- `attendance_date` (Date, Required): Attendance Date
- `shift` (Link, Options: Shift Type): Shift
- `late_entry` (Check, Default: 0): Late Entry
- `early_exit` (Check, Default: 0): Early Exit
- `half_day_status` (Select, Options: 
Present
Absent): Status for Other Half (Depends on: eval:doc.status=="Half Day";)
- `modify_half_day_status` (Check, Default: 0): modify_half_day_status

### Read-only Fields

- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `working_hours` (Float, Read-only): Working Hours (Depends on: working_hours)
- `leave_application` (Link, Read-only, Options: Leave Application): Leave Application
- `company` (Link, Required, Read-only, Options: Company): Company (Fetched from: employee.company)
- `department` (Link, Read-only, Options: Department): Department (Fetched from: employee.department)
- `attendance_request` (Link, Read-only, Options: Attendance Request): Attendance Request
- `in_time` (Datetime, Read-only): In Time (Depends on: shift)
- `out_time` (Datetime, Read-only): Out Time (Depends on: shift)

## Workflow

### Step 1: Create Document
1. Go to HRMS → Attendance → New
2. Fill required fields:
   - `naming_series`: Series
   - `employee`: Employee
   - `status`: Status
   - `attendance_date`: Attendance Date

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: naming_series:

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
