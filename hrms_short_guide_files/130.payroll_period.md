# Payroll Period

## Overview
Payroll Period DocType for HRMS Payroll.

## Fields

### Main Fields

- `company` (Link, Required, Options: Company): Company
- `start_date` (Date, Required): Start Date
- `end_date` (Date, Required): End Date

### Child Tables

- `periods` (Table): Payroll Periods
  - Options: Payroll Period Date

## Workflow

### Step 1: Create Document
1. Go to HRMS Payroll → Payroll Period → New
2. Fill required fields:
   - `company`: Company
   - `start_date`: Start Date
   - `end_date`: End Date

### Step 2: Save/Submit
1. Review all fields
2. Save document

### Naming
- Auto-naming: Prompt

## Important Notes
- Document must be saved to be effective
