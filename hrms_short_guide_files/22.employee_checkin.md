# Employee Checkin

## Overview
Employee Checkin DocType for HRMS.

## Fields

### Main Fields

- `employee` (Link, Required, Options: Employee): Employee
- `log_type` (Select, Options: 
IN
OUT): Log Type
- `time` (Datetime, Required, Default: Now): Time
- `device_id` (Data): Location / Device ID
- `skip_auto_attendance` (Check, Default: 0): Skip Auto Attendance
- `shift_start` (Datetime): Shift Start
- `shift_end` (Datetime): Shift End
- `shift_actual_start` (Datetime): Shift Actual Start
- `shift_actual_end` (Datetime): Shift Actual End

### Read-only Fields

- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `shift` (Link, Read-only, Options: Shift Type): Shift
- `attendance` (Link, Read-only, Options: Attendance): Attendance Marked
- `geolocation` (Geolocation, Read-only): Geolocation
- `latitude` (Float, Read-only): Latitude
- `longitude` (Float, Read-only): Longitude
- `offshift` (Check, Read-only, Default: 0): Off-shift

## Workflow

### Step 1: Create Document
1. Go to HRMS → Employee Checkin → New
2. Fill required fields:
   - `employee`: Employee
   - `time`: Time

### Step 2: Save/Submit
1. Review all fields
2. Save document

### Naming
- Auto-naming: EMP-CKIN-.MM.-.YYYY.-.######

## Important Notes
- Document must be saved to be effective
