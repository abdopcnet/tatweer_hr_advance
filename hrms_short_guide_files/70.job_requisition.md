# Job Requisition

## Overview
Job Requisition DocType for HRMS.

## Fields

### Main Fields

- `designation` (Link, Required, Options: Designation): Designation
- `no_of_positions` (Int, Required): No of. Positions
- `expected_compensation` (Currency, Required): Expected Compensation
- `status` (Select, Required, Options: Pending
Open & Approved
Rejected
Filled
On Hold
Cancelled): Status
- `company` (Link, Required, Options: Company): Company
- `requested_by` (Link, Required, Options: Employee): Requested By
- `posting_date` (Date, Required): Posting Date
- `expected_by` (Date): Expected By
- `completed_on` (Date): Completed On (Depends on: eval:doc.status=="Filled")
- `description` (Text Editor, Required): Job Description (Fetched from: designation.description)
- `naming_series` (Select, Options: HR-HIREQ-): Naming Series
- `reason_for_requesting` (Text): Reason for Requesting
- `department` (Link, Options: Department): Department

### Read-only Fields

- `requested_by_name` (Data, Read-only): Requested By (Name) (Fetched from: requested_by.employee_name)
- `requested_by_dept` (Link, Read-only, Options: Department): Department (Fetched from: requested_by.department)
- `requested_by_designation` (Link, Read-only, Options: Designation): Designation (Fetched from: requested_by.designation)
- `time_to_fill` (Duration, Read-only): Time to Fill

## Workflow

### Step 1: Create Document
1. Go to HRMS → Job Requisition → New
2. Fill required fields:
   - `designation`: Designation
   - `no_of_positions`: No of. Positions
   - `expected_compensation`: Expected Compensation
   - `status`: Status
   - `company`: Company
   - `requested_by`: Requested By
   - `posting_date`: Posting Date
   - `description`: Job Description

### Step 2: Save/Submit
1. Review all fields
2. Save document
