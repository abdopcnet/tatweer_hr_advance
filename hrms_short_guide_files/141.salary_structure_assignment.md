# Salary Structure Assignment

## Overview
Salary Structure Assignment DocType for HRMS Payroll.

## Fields

### Main Fields

- `employee` (Link, Required, Options: Employee): Employee
- `salary_structure` (Link, Required, Options: Salary Structure): Salary Structure (Fetched from: grade.default_salary_structure)
- `from_date` (Date, Required): From Date
- `company` (Link, Required, Options: Company): Company (Fetched from: employee.company)
- `base` (Currency): Base (Fetched from: grade.default_base_pay)
- `variable` (Currency): Variable
- `income_tax_slab` (Link, Options: Income Tax Slab): Income Tax Slab (Depends on: salary_structure)
- `payroll_payable_account` (Link, Options: Account): Payroll Payable Account (Depends on: employee)
- `tax_deducted_till_date` (Currency): Tax Deducted Till Date
- `taxable_earnings_till_date` (Currency): Taxable Earnings Till Date

### Read-only Fields

- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `department` (Link, Read-only, Options: Department): Department (Fetched from: employee.department)
- `designation` (Link, Read-only, Options: Designation): Designation (Fetched from: employee.designation)
- `currency` (Link, Required, Read-only, Options: Currency): Currency (Fetched from: salary_structure.currency) (Depends on: eval:(doc.docstatus==1 || doc.salary_structure))
- `grade` (Link, Read-only, Options: Employee Grade): Grade (Fetched from: employee.grade)

### Child Tables

- `payroll_cost_centers` (Table): Cost Centers
  - Options: Employee Cost Center

## Workflow

### Step 1: Create Document
1. Go to HRMS Payroll → Salary Structure Assignment → New
2. Fill required fields:
   - `employee`: Employee
   - `salary_structure`: Salary Structure
   - `from_date`: From Date
   - `company`: Company

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: HR-SSA-.YY.-.MM.-.#####

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
