# Training Feedback

## Overview
Training Feedback DocType for HRMS.

## Fields

### Main Fields

- `employee` (Link, Required, Options: Employee): Employee
- `employee_name` (Read Only): Employee Name (Fetched from: employee.employee_name)
- `training_event` (Link, Required, Options: Training Event): Training Event
- `feedback` (Text, Required): Feedback

### Read-only Fields

- `department` (Link, Read-only, Options: Department): Department (Fetched from: employee.department)
- `course` (Data, Read-only): Course (Fetched from: training_event.course)
- `event_name` (Data, Read-only): Event Name (Fetched from: training_event.event_name)
- `trainer_name` (Data, Read-only): Trainer Name (Fetched from: training_event.trainer_name)

## Workflow

### Step 1: Create Document
1. Go to HRMS → Training Feedback → New
2. Fill required fields:
   - `employee`: Employee
   - `training_event`: Training Event
   - `feedback`: Feedback

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: HR-TRF-.YYYY.-.#####

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
