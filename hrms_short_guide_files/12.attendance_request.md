# Attendance Request

## Overview
Attendance Request DocType for HRMS.

## Fields

### Main Fields

- `employee` (Link, Required, Options: Employee): Employee
- `company` (Link, Required, Options: Company): Company (Fetched from: employee.company)
- `from_date` (Date, Required): From Date
- `to_date` (Date, Required): To Date
- `half_day` (Check, Default: 0): Half Day
- `half_day_date` (Date): Half Day Date (Depends on: half_day)
- `reason` (Select, Required, Options: Work From Home
On Duty): Reason
- `explanation` (Small Text): Explanation
- `shift` (Link, Options: Shift Type): Shift
- `include_holidays` (Check, Default: 0): Include Holidays

### Read-only Fields

- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `department` (Link, Read-only, Options: Department): Department (Fetched from: employee.department)

## Workflow

### Step 1: Create Document
1. Go to HRMS → Attendance Request → New
2. Fill required fields:
   - `employee`: Employee
   - `company`: Company
   - `from_date`: From Date
   - `to_date`: To Date
   - `reason`: Reason

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: HR-ARQ-.YY.-.MM.-.#####

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
