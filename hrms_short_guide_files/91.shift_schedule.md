# Shift Schedule

## Overview
Shift Schedule DocType for HRMS.

## Fields

### Main Fields

- `frequency` (Select, Required, Options: Every Week
Every 2 Weeks
Every 3 Weeks
Every 4 Weeks): Frequency
- `shift_type` (Link, Required, Options: Shift Type): Shift Type

### Child Tables

- `repeat_on_days` (Table): Repeat On Days
  - Options: Assignment Rule Day

## Workflow

### Step 1: Create Document
1. Go to HRMS → Shift Schedule → New
2. Fill required fields:
   - `frequency`: Frequency
   - `shift_type`: Shift Type

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: prompt

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
