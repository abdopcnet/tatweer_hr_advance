# Employee Referral

## Overview
Employee Referral DocType for HRMS.

## Fields

### Main Fields

- `first_name` (Data, Required): First Name 
- `last_name` (Data, Required): Last Name
- `contact_no` (Data): Contact No.
- `current_employer` (Data): Current Employer 
- `date` (Date, Required): Date
- `current_job_title` (Data): Current Job Title
- `resume` (Attach): Resume
- `work_references` (Text Editor): Work References
- `for_designation` (Link, Required, Options: Designation): For Designation 
- `email` (Data, Required): Email
- `is_applicable_for_referral_bonus` (Check, Default: 1): Is Applicable for Referral Bonus
- `qualification_reason` (Text Editor): Why is this Candidate Qualified for this Position?
- `referrer` (Link, Required, Options: Employee): Referrer
- `resume_link` (Data): Resume Link

### Read-only Fields

- `full_name` (Data, Read-only): Full Name
- `status` (Select, Required, Read-only, Options: Pending
In Process
Accepted
Rejected): Status
- `department` (Link, Read-only, Options: Department): Department (Fetched from: employee.department)
- `referrer_name` (Data, Read-only): Referrer Name (Fetched from: referrer.employee_name)
- `referral_payment_status` (Select, Read-only, Options: 
Unpaid
Paid): Referral Bonus Payment Status

## Workflow

### Step 1: Create Document
1. Go to HRMS → Employee Referral → New
2. Fill required fields:
   - `first_name`: First Name 
   - `last_name`: Last Name
   - `date`: Date
   - `for_designation`: For Designation 
   - `email`: Email
   - `referrer`: Referrer

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: format:HR-REF-{####}

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
