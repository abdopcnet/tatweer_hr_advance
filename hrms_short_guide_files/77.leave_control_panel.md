# Leave Control Panel

## Overview
Leave Control Panel DocType for HRMS.

## Fields

### Main Fields

- `company` (Link, Options: Company): Company
- `employment_type` (Link, Options: Employment Type): Employment Type
- `branch` (Link, Options: Branch): Branch
- `department` (Link, Options: Department): Department
- `designation` (Link, Options: Designation): Designation
- `employee_grade` (Link, Options: Employee Grade): Employee Grade
- `from_date` (Date): From Date (Depends on: eval:doc.dates_based_on != 'Joining Date')
- `to_date` (Date): To Date
- `leave_policy` (Link, Options: Leave Policy): Leave Policy (Depends on: eval:doc.allocate_based_on_leave_policy )
- `leave_type` (Link, Options: Leave Type): Leave Type (Depends on: eval:!doc.allocate_based_on_leave_policy)
- `carry_forward` (Check, Default: 1): Carry Forward
- `no_of_days` (Float): New Leaves Allocated (In Days) (Depends on: eval:!doc.allocate_based_on_leave_policy)
- `dates_based_on` (Select, Options: Leave Period
Joining Date
Custom Range, Default: Leave Period): Dates Based On
- `leave_period` (Link, Options: Leave Period): Leave Period (Depends on: eval:doc.dates_based_on == 'Leave Period')
- `allocate_based_on_leave_policy` (Check, Default: 1): Allocate Based On Leave Policy

## Workflow

### Step 1: Create Document
1. Go to HRMS → Leave Control Panel → New

### Step 2: Save/Submit
1. Review all fields
2. Save document

## Important Notes
- Document must be saved to be effective
