# Appointment Letter Template

## Overview
Appointment Letter Template DocType for HRMS.

## Fields

### Main Fields

- `introduction` (Long Text, Required): Introduction
- `closing_notes` (Text): Closing Notes
- `template_name` (Data, Required): Template Name

### Child Tables

- `terms` (Table): Terms
  - Options: Appointment Letter content

## Workflow

### Step 1: Create Document
1. Go to HRMS → Appointment Letter Template → New
2. Fill required fields:
   - `introduction`: Introduction
   - `template_name`: Template Name

### Step 2: Save/Submit
1. Review all fields
2. Save document

### Naming
- Auto-naming: field:template_name

## Important Notes
- Document must be saved to be effective
