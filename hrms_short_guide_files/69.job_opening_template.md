# Job Opening Template

## Overview
Job Opening Template DocType for HRMS.

## Fields

### Main Fields

- `department` (Link, Options: Department): Department
- `employment_type` (Link, Options: Employment Type): Employment Type
- `location` (Link, Options: Branch): Location
- `template_title` (Data, Required): Template Title
- `description` (Text Editor): Description

## Workflow

### Step 1: Create Document
1. Go to HRMS → Job Opening Template → New
2. Fill required fields:
   - `template_title`: Template Title

### Step 2: Save/Submit
1. Review all fields
2. Save document

### Naming
- Auto-naming: field:template_title

## Important Notes
- Document must be saved to be effective
