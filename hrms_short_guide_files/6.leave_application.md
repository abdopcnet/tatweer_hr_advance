# Leave Application

## Overview
Leave Application DocType for HRMS.

## Fields

### Main Fields

- `naming_series` (Select, Required, Options: HR-LAP-.YYYY.-): Series
- `employee` (Link, Required, Options: Employee): Employee
- `employee_name` (Data, Read-only): Employee Name
- `leave_type` (Link, Required, Options: Leave Type): Leave Type
- `department` (Link, Read-only, Options: Department): Department
- `leave_balance` (Float, Read-only): Leave Balance Before Application
- `from_date` (Date, Required): From Date
- `to_date` (Date, Required): To Date
- `half_day` (Check, Default: 0): Half Day
- `half_day_date` (Date): Half Day Date
- `total_leave_days` (Float, Read-only): Total Leave Days
- `description` (Small Text): Reason
- `leave_approver` (Link, Options: User): Leave Approver
- `leave_approver_name` (Data, Read-only): Leave Approver Name
- `status` (Select, Required, Options: Open
Approved
Rejected
Cancelled, Default: Open): Status
- `posting_date` (Date, Required, Default: Today): Posting Date
- `company` (Link, Required, Read-only, Options: Company): Company
- `follow_via_email` (Check, Default: 1): Follow via Email
- `salary_slip` (Link, Options: Salary Slip): Salary Slip
- `letter_head` (Link, Options: Letter Head): Letter Head
- `color` (Color): Color
- `amended_from` (Link, Read-only, Options: Leave Application): Amended From

## Workflow

### Step 1: Create Document
1. Go to HRMS → Leave Application → New
2. Fill required fields:
   - `naming_series`: Series
   - `employee`: Employee
   - `leave_type`: Leave Type
   - `from_date`: From Date
   - `to_date`: To Date

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
