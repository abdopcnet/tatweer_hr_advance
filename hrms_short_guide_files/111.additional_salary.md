# Additional Salary

## Overview
Additional Salary DocType for HRMS Payroll.

## Fields

### Main Fields

- `naming_series` (Select, Required, Options: HR-ADS-.YY.-.MM.-): Series
- `employee` (Link, Required, Options: Employee): Employee
- `salary_component` (Link, Required, Options: Salary Component): Salary Component
- `amount` (Currency, Required): Amount
- `overwrite_salary_structure_amount` (Check, Default: 1): Overwrite Salary Structure Amount
- `deduct_full_tax_on_selected_payroll_date` (Check, Default: 0): Deduct Full Tax on Selected Payroll Date
- `payroll_date` (Date): Payroll Date (Depends on: eval:(doc.is_recurring==0))
- `company` (Link, Required, Options: Company): Company
- `is_recurring` (Check, Default: 0): Is Recurring
- `from_date` (Date): From Date (Depends on: eval:(doc.is_recurring==1))
- `to_date` (Date): To Date (Depends on: eval:(doc.is_recurring==1))
- `disabled` (Check, Default: 0): Disabled (Depends on: eval:doc.is_recurring)

### Read-only Fields

- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `department` (Link, Read-only, Options: Department): Department (Fetched from: employee.department)
- `type` (Data, Read-only): Salary Component Type (Fetched from: salary_component.type)
- `ref_doctype` (Link, Read-only, Options: DocType): Reference Document Type
- `ref_docname` (Dynamic Link, Read-only): Reference Document
- `currency` (Link, Required, Read-only, Options: Currency): Currency (Depends on: eval:(doc.docstatus==1 || doc.employee))

## Workflow

### Step 1: Create Document
1. Go to HRMS Payroll → Additional Salary → New
2. Fill required fields:
   - `naming_series`: Series
   - `employee`: Employee
   - `salary_component`: Salary Component
   - `amount`: Amount
   - `company`: Company

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document
