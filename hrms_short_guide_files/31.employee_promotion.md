# Employee Promotion

## Overview
Employee Promotion DocType for HRMS.

## Fields

### Main Fields

- `employee` (Link, Required, Options: Employee): Employee
- `promotion_date` (Date, Required): Promotion Date
- `company` (Link, Options: Company): Company (Fetched from: employee.company)
- `current_ctc` (Currency): Current CTC (Fetched from: employee.ctc)
- `revised_ctc` (Currency): Revised CTC (Depends on: current_ctc)

### Read-only Fields

- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `department` (Link, Read-only, Options: Department): Department (Fetched from: employee.department)
- `salary_currency` (Link, Read-only, Options: Currency): Salary Currency (Fetched from: employee.salary_currency)

### Child Tables

- `promotion_details` (Table): promotion_details
  - Options: Employee Property History

## Workflow

### Step 1: Create Document
1. Go to HRMS → Employee Promotion → New
2. Fill required fields:
   - `employee`: Employee
   - `promotion_date`: Promotion Date

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: HR-EMP-PRO-.YYYY.-.#####

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
