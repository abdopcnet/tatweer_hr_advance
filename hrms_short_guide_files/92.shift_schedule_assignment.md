# Shift Schedule Assignment

## Overview
Shift Schedule Assignment DocType for HRMS.

## Fields

### Main Fields

- `enabled` (Check, Default: 1): Enabled
- `create_shifts_after` (Date): Create Shifts After (Depends on: eval:doc.enabled)
- `employee` (Link, Required, Options: Employee): Employee
- `shift_status` (Select, Options: Active
Inactive, Default: Active): Shift Status
- `shift_schedule` (Link, Required, Options: Shift Schedule): Shift Schedule
- `shift_location` (Link, Options: Shift Location): Shift Location

### Read-only Fields

- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `company` (Link, Required, Read-only, Options: Company): Company (Fetched from: employee.company)

## Workflow

### Step 1: Create Document
1. Go to HRMS → Shift Schedule Assignment → New
2. Fill required fields:
   - `employee`: Employee
   - `shift_schedule`: Shift Schedule

### Step 2: Save/Submit
1. Review all fields
2. Save document

### Naming
- Auto-naming: HR-SHSA-.YY.-.MM.-.#####

## Important Notes
- Document must be saved to be effective
