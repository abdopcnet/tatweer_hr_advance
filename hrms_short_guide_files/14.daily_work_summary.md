# Daily Work Summary

## Overview
Daily Work Summary DocType for HRMS.

## Fields

### Read-only Fields

- `daily_work_summary_group` (Link, Read-only, Options: Daily Work Summary Group): Daily Work Summary Group
- `status` (Select, Read-only, Options: Open
Sent, Default: Open): Status
- `email_sent_to` (Code, Read-only): Email Sent To

## Workflow

### Step 1: Create Document
1. Go to HRMS → Daily Work Summary → New

### Step 2: Save/Submit
1. Review all fields
2. Save document

## Important Notes
- Document must be saved to be effective
