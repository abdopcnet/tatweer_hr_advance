# Employee Separation

## Overview
Employee Separation DocType for HRMS.

## Fields

### Main Fields

- `employee` (Link, Required, Options: Employee): Employee
- `notify_users_by_email` (Check, Default: 0): Notify users by email
- `employee_separation_template` (Link, Options: Employee Separation Template): Employee Separation Template
- `company` (Link, Required, Options: Company): Company (Fetched from: employee.company)
- `exit_interview` (Text Editor): Exit Interview Summary
- `boarding_begins_on` (Date, Required): Separation Begins On

### Read-only Fields

- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `resignation_letter_date` (Date, Read-only): Resignation Letter Date (Fetched from: employee.resignation_letter_date)
- `boarding_status` (Select, Read-only, Options: Pending
In Process
Completed, Default: Pending): Status
- `project` (Link, Read-only, Options: Project): Project
- `department` (Link, Read-only, Options: Department): Department (Fetched from: employee.department)
- `designation` (Link, Read-only, Options: Designation): Designation (Fetched from: employee.designation)
- `employee_grade` (Link, Read-only, Options: Employee Grade): Employee Grade (Fetched from: employee.grade)

### Child Tables

- `activities` (Table): Activities
  - Options: Employee Boarding Activity

## Workflow

### Step 1: Create Document
1. Go to HRMS → Employee Separation → New
2. Fill required fields:
   - `employee`: Employee
   - `company`: Company
   - `boarding_begins_on`: Separation Begins On

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: HR-EMP-SEP-.YYYY.-.#####

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
