# Salary Structure

## Overview
Salary Structure DocType for HRMS Payroll.

## Fields

### Main Fields

- `company` (Link, Required, Options: Company): Company
- `letter_head` (Link, Options: Letter Head): Letter Head (Fetched from: company.default_letter_head)
- `is_active` (Select, Required, Options: 
Yes
No, Default: Yes): Is Active
- `payroll_frequency` (Select, Options: 
Monthly
Fortnightly
Bimonthly
Weekly
Daily, Default: Monthly): Payroll Frequency (Depends on: eval:doc.salary_slip_based_on_timesheet == 0)
- `salary_slip_based_on_timesheet` (Check, Default: 0): Salary Slip Based on Timesheet
- `salary_component` (Link, Options: Salary Component): Salary Component
- `hour_rate` (Currency): Hour Rate
- `leave_encashment_amount_per_day` (Currency): Leave Encashment Amount Per Day
- `max_benefits` (Currency): Max Benefits (Amount)
- `mode_of_payment` (Link, Options: Mode of Payment): Mode of Payment
- `payment_account` (Link, Options: Account): Payment Account
- `currency` (Link, Required, Options: Currency): Currency

### Read-only Fields

- `is_default` (Select, Read-only, Options: Yes
No, Default: No): Is Default
- `total_earning` (Currency, Read-only): Total Earning
- `total_deduction` (Currency, Read-only): Total Deduction
- `net_pay` (Currency, Read-only): Net Pay

### Child Tables

- `earnings` (Table): Earnings
  - Options: Salary Detail
- `deductions` (Table): Deductions
  - Options: Salary Detail

## Workflow

### Step 1: Create Document
1. Go to HRMS Payroll → Salary Structure → New
2. Fill required fields:
   - `company`: Company
   - `is_active`: Is Active
   - `currency`: Currency

### Step 2: Save/Submit
1. Review all fields
2. Save document
3. Submit document

### Naming
- Auto-naming: Prompt

## Important Notes
- Document must be saved to be effective
- Document must be submitted to be effective
