# Goal

## Overview
Goal DocType for HRMS.

## Fields

### Main Fields

- `employee` (Link, Required, Options: Employee): Employee
- `start_date` (Date, Required): Start Date (Fetched from: appraisal_cycle.start_date) (Depends on: employee)
- `progress` (Percent): Progress
- `description` (Text Editor): Description
- `is_group` (Check, Default: 0): Is Group
- `old_parent` (Link, Options: Goal): Old Parent
- `parent_goal` (Link, Options: Goal): Parent Goal (Depends on: employee)
- `kra` (Link, Options: KRA): KRA (Fetched from: parent_goal.kra) (Depends on: employee)
- `appraisal_cycle` (Link, Options: Appraisal Cycle): Appraisal Cycle (Fetched from: parent_goal.appraisal_cycle) (Depends on: employee)
- `end_date` (Date): End Date (Fetched from: appraisal_cycle.end_date) (Depends on: employee)
- `goal_name` (Data, Required): Goal

### Read-only Fields

- `status` (Select, Read-only, Options: 
Pending
In Progress
Completed
Archived
Closed, Default: Pending): Status
- `lft` (Int, Read-only): Left
- `rgt` (Int, Read-only): Right
- `employee_name` (Data, Read-only): Employee Name (Fetched from: employee.employee_name)
- `user` (Data, Read-only): User (Fetched from: employee.user_id)
- `company` (Link, Read-only, Options: Company): Company (Fetched from: employee.company)

## Workflow

### Step 1: Create Document
1. Go to HRMS → Goal → New
2. Fill required fields:
   - `employee`: Employee
   - `start_date`: Start Date
   - `goal_name`: Goal

### Step 2: Save/Submit
1. Review all fields
2. Save document

### Naming
- Auto-naming: format:HR-GOAL-{YYYY}-{####}

## Important Notes
- Document must be saved to be effective
