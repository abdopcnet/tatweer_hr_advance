# Appointment Letter

## Overview
Appointment Letter DocType for HRMS.

## Fields

### Main Fields

- `appointment_date` (Date, Required): Appointment Date
- `appointment_letter_template` (Link, Required, Options: Appointment Letter Template): Appointment Letter Template
- `introduction` (Long Text, Required): Introduction (Fetched from: appointment_letter_template.introduction)
- `job_applicant` (Link, Required, Options: Job Applicant): Job Applicant
- `company` (Link, Required, Options: Company): Company
- `closing_notes` (Text): Closing Notes

### Read-only Fields

- `applicant_name` (Data, Required, Read-only): Applicant Name (Fetched from: job_applicant.applicant_name)

### Child Tables

- `terms` (Table): Terms
  - Options: Appointment Letter content

## Workflow

### Step 1: Create Document
1. Go to HRMS → Appointment Letter → New
2. Fill required fields:
   - `appointment_date`: Appointment Date
   - `appointment_letter_template`: Appointment Letter Template
   - `introduction`: Introduction
   - `job_applicant`: Job Applicant
   - `company`: Company

### Step 2: Save/Submit
1. Review all fields
2. Save document

### Naming
- Auto-naming: HR-APP-LETTER-.#####

## Important Notes
- Document must be saved to be effective
