# Hr Advance

![Version](https://img.shields.io/badge/version-2.1.2026-blue)

Advanced HR management features for ERPNext.

## Features

### Payroll & Salary
- Salary Slip & Payroll Entry overrides
- Designation Type with salary details
- Additional Salary component summation
- Employee base salary calculation

### Employee Management
- Employee experience calculation (daily scheduled task)
- Bank Branch management
- Employee Training request workflow

### Loans & Advances
- Loan amount calculation API
- Employee Advance bank filter

### Leave Management
- Bulk Leave Allocation (multiple employees)
- Leave validation (no earlier dates)
- Automatic carry forward calculation

### Travel & Expenses
- Travel Request to Expense Claim conversion

### Reports
- Attendance Report (late entry, early exit tracking)
- Exit Permission Report
